Laporan Teori Pemrograman Web B

  1. Pertanyaan seputar Tag

    a. Apa yang dimaksud dengan tag pada HTML?
    Tag adalah penanda dalam HTML yang digunakan untuk membuat struktur halaman web.

    b. Apa itu pembuka dan penutup tag?
    Tag pembuka seperti <p> dan penutup seperti </p>. Tidak semua tag memiliki penutup, contohnya <img>.

  2. Pertanyaan seputar Element

    a. Apa yang dimaksud dengan element?
    Element adalah gabungan tag pembuka, isi, dan tag penutup.

    b. Tagname, attribute, content?
    Tagname adalah nama tag, attribute adalah properti tambahan, dan content adalah isi.

    c. Apakah elemen dapat ditumpuk?
    Ya, disebut nested element. Jika salah, struktur bisa rusak.

  3. Pertanyaan seputar Anchor

    a. Apa itu anchor?
    Anchor adalah tag untuk membuat hyperlink.

    b. Cara kerja anchor?
    Menggunakan atribut href untuk menentukan tujuan link.

  4. Pertanyaan seputar Multimedia

    Text:
    Tag: <b>, <i>, <u> untuk format teks.
    Struktur: <p>, <h1>.

    Image:
    Gambar tidak tampil bisa karena path salah.
    Teks muncul karena atribut alt.

    Audio:
    Bisa dari web menggunakan tag <audio> dan source.

    Video:
    Autoplay sering diblokir, harus gunakan atribut muted.
